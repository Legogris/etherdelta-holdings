EtherDelta token balance checker


Lost track of your funds at EtherDelta? Worry not - just put in your address and this tool will return all amounts that address has deposited into the EtherDelta smart contract.

Note that this only includes tokens officially listed at ED at the current moment. If you have deposited other tokens (such as DAT), these will not show up here.

Requires a modern browser (tested in Firefox 57.0.4)
